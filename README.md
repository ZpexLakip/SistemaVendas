# Sistema de Vendas

## Descrição

Projeto acadêmico desenvolvido com o objetivo de aplicar conceitos de Programação Orientada a Objetos (POO) em Java, simulando um sistema básico de vendas. O sistema aborda operações essenciais como gerenciamento de produtos, registro de vendas e organização de dados.

Devido à estrutura do projeto, os arquivos fonte estão compactados em um arquivo `.zip`.

## Estrutura do Projeto

* **Vendas.ATV.zip**: Arquivo compactado contendo todo o código-fonte do sistema, incluindo classes e implementação das funcionalidades.

> Para acessar o conteúdo, é necessário extrair o arquivo `.zip`.

## Funcionalidades Esperadas

O sistema pode incluir funcionalidades como:

* Cadastro de produtos
* Registro de vendas
* Associação entre produtos e vendas
* Manipulação de dados via classes Java

## Objetivo

Consolidar conhecimentos em:

* Programação Orientada a Objetos
* Estruturação de projetos em Java
* Organização e encapsulamento de dados
* Lógica de sistemas comerciais simples

## Tecnologias Utilizadas

* Java

## Como Executar

1. Faça o download do repositório
2. Extraia o arquivo **Vendas.ATV.zip**
3. Abra o projeto em uma IDE Java (IntelliJ, Eclipse ou VS Code)
4. Execute a classe principal (geralmente `Main.java`)
